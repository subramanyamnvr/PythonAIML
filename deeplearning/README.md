# Deep Learning

## Overview
Deep learning implementations using neural networks, covering CNNs, RNNs, and transformer architectures.

## Topics Covered
- Neural network fundamentals
- Convolutional Neural Networks (CNNs)
- Recurrent Neural Networks (RNNs)
- Transformer models

## Prerequisites
- Machine learning basics
- Python programming
- Calculus and linear algebra

## Subfolders
- **neural-networks/** - Basic neural network concepts
- **cnn/** - Convolutional neural networks
- **rnn/** - Recurrent neural networks
- **transformers/** - Transformer architectures

## Learning Path
1. Start with neural-networks/ basics
2. Learn cnn/ for vision tasks
3. Explore rnn/ for sequential data
4. Study transformers/ for modern NLP
