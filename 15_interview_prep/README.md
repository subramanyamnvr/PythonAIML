# Interview Prep Hub

This section is the high-speed revision area of the repository.

## Recommended Contents

- `01_python_questions`: Python interview Q&A
- `02_sql_and_stats_questions`: SQL, probability, statistics basics
- `03_ml_questions`: ML concepts, algorithms, metrics, debugging
- `04_dl_questions`: neural networks, training, architecture questions
- `05_genai_questions`: LLMs, RAG, agents, embeddings, evals
- `06_system_design_questions`: ML/GenAI system design prompts
- `07_case_studies`: product, modeling, and debugging scenarios
- `08_revision_cheatsheets`: fast revision notes

## Best Practice

For each question, capture:

- short answer
- deeper answer
- example
- common follow-up

That makes revision much more effective than storing only definitions.
