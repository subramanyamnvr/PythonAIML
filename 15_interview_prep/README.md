# Interview Prep Hub

This section is the high-speed revision area of the repository.

## Current Sections

- `01_python_questions`: Python interview Q&A, DSA prompts, and implementation-use-case notes
- `02_sql_and_stats_questions`: SQL, probability, statistics, experiments, and analytics-oriented interview material
- `03_ml_questions`: ML concepts, algorithms, metrics, debugging, and tradeoff questions
- `04_dl_questions`: neural networks, optimization, architecture, and training-system questions
- `05_genai_questions`: LLMs, RAG, agents, embeddings, evals, guardrails, and multimodal system questions
- `06_system_design_questions`: ML and GenAI system design prompts, architecture breakdowns, and deployment tradeoffs
- `07_case_studies`: product, modeling, experimentation, and debugging scenarios
- `08_revision_cheatsheets`: fast revision notes and condensed comparison sheets

## How To Use It

- Keep each note inside the most relevant numbered section instead of dropping files at the root.
- Start by building `01` through `05`, then use `06` and `07` for deeper interview rounds.
- Use `08_revision_cheatsheets` for the shortest high-speed review material right before interviews.

## Best Practice

For each question, capture:

- short answer
- deeper answer
- example
- common follow-up

That makes revision much more effective than storing only definitions.
