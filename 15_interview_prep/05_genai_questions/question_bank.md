# GenAI Question Bank

## 1. What makes a RAG system fail?

- weak chunking
- poor retrieval
- irrelevant context stuffing
- no citations or evals
- bad prompt grounding

## 2. What is the difference between fine-tuning and prompt engineering?

- Prompting changes instructions at inference time.
- Fine-tuning changes model behavior through training.

## 3. What are evals for?

- They measure quality, catch regressions, and make prompt or retrieval changes safer.

## 4. Why do guardrails matter?

- They reduce unsafe outputs, policy violations, prompt injection risk, and bad tool calls.

## 5. What is agent memory?

- It is the combination of short-term state and longer-lived stored context that helps an agent continue work coherently.
