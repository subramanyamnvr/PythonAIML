# ML Question Bank

## 1. Bias vs variance

- High bias means the model is too simple and underfits.
- High variance means the model is too sensitive and overfits.

## 2. What is data leakage?

- Leakage happens when training gets access to information it would not have at prediction time.

## 3. How do you choose between tree models and linear models?

- Linear models win when interpretability and stable behavior matter.
- Tree ensembles win when interactions and nonlinearity dominate.

## 4. Why do train, validation, and test all matter?

- Train fits the model.
- Validation tunes choices.
- Test estimates final generalization.

## 5. What do you check when a model suddenly degrades?

- upstream data changes
- target drift
- feature null rates
- class balance shifts
- threshold mismatch
