# CV Capstone

## Focus
An end-to-end computer-vision project folder that ties together detection, tracking, segmentation, or OCR into one deliverable.

## Included Here
- `capstone_pipeline.py`: a starter pipeline showing how a capstone flow can be organized
- `README.md`: the quick-start guide for this topic folder

## Run
`python capstone_pipeline.py`

## Expand With
- a detection-plus-tracking analytics project
- an OCR or segmentation pipeline with reporting and deployment hooks
