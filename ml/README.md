# Machine Learning

## Overview
Comprehensive collection of machine learning algorithms, from traditional methods to modern ensemble techniques.

## Topics Covered
- Supervised learning (regression, classification)
- Unsupervised learning (clustering, dimensionality reduction)
- Ensemble methods (bagging, boosting)

## Prerequisites
- Python programming
- Statistics and probability
- Linear algebra

## Subfolders
- **supervised/** - Regression and classification algorithms
- **unsupervised/** - Clustering and dimensionality reduction
- **ensemble/** - Ensemble learning methods

## Learning Path
1. Master supervised/ algorithms
2. Explore unsupervised/ techniques
3. Learn ensemble/ methods for better performance
