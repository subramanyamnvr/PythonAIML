# Python AI/ML Master Repository

A comprehensive collection of Python, Artificial Intelligence, Machine Learning, Computer Vision, Deep Learning, and Generative AI resources, projects, and tutorials.

## 📁 Repository Structure

- **python/** - Python fundamentals, libraries, and projects
- **ai/** - Artificial Intelligence concepts and algorithms
- **ml/** - Machine Learning algorithms (supervised, unsupervised, ensemble)
- **vision/** - Computer vision and image processing
- **deeplearning/** - Deep learning with neural networks, CNNs, RNNs, Transformers
- **genai/** - Generative AI, LLMs, prompt engineering, fine-tuning

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/subramanyamnvr/PythonAIML.git
   cd PythonAIML
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Explore the folders and run the notebooks!

## 📚 Learning Path

1. Start with `python/basics/` for Python fundamentals
2. Move to `ml/supervised/` for basic ML algorithms
3. Explore `deeplearning/` for neural networks
4. Dive into `genai/` for modern AI applications
