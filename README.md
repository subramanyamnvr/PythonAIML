# Python + AI/ML/Deep Learning/GenAI Repo

This repository is a single study and revision workspace for Python, data libraries, NLP, machine learning, deep learning, computer vision, GenAI, MLOps, projects, and interview prep.

Several course-heavy folders have been reorganized for easier browsing:

- numbered notebooks are the main entry point where a flat layer exists
- original support folders stay in place when notebooks depend on local data, helpers, or models
- machine learning topics are sorted into clean numbered folders

- `00_repo_guide`: repo navigation docs and study planning

- `04-Matplotlib`: plotting notebooks
- `05-Seaborn`: seaborn notebooks
- `06-NLPComplete`: flat NLP notebook path with datasets and README

- `10_resources`: future resources, references, templates, and datasets

- `12-mlops_and_deployment`: scaffold for production and deployment study
