# Python + AI/ML/Deep Learning/GenAI Master Repo

This repository is designed to be a single, interview-friendly learning and revision hub for:

- Python fundamentals
- Important Python libraries
- Math for AI/ML
- Machine Learning
- Deep Learning
- Generative AI
- MLOps and deployment
- Projects and interview preparation

## Goals

- Learn concepts in a structured order
- Keep theory, code, notes, and interview prep in one place
- Build a reusable portfolio of examples and mini-projects
- Make revision fast before interviews



## Recommended Learning Flow

1. `01_python_fundamentals`
2. `02_python_libraries`
3. `03_math_for_ai_ml`
4. `04_machine_learning`
5. `05_deep_learning`
6. `06_genai`
7. `07_mlops_and_deployment`
8. `08_projects`
9. `09_interview_prep`

## Top-Level Structure

- `00_repo_guide`: how to use this repo
- `01_python_fundamentals`: core Python for coding rounds and practical work
- `02_python_libraries`: high-impact libraries used in real projects
- `03_math_for_ai_ml`: math foundations required for ML interviews
- `04_machine_learning`: end-to-end ML concepts and algorithms
- `05_deep_learning`: neural networks, frameworks, and model families
- `06_genai`: modern LLM, RAG, agents, evals, and system design
- `07_mlops_and_deployment`: production and deployment fundamentals
- `08_projects`: portfolio-ready hands-on work
- `09_interview_prep`: direct question banks, case studies, cheatsheets
- `10_resources`: books, papers, datasets, templates
- `11_sandbox`: scratch area for experiments
- `12_progress_tracking`: study plans and progress logs
