# Time Series Forecasting

## Focus
Classical forecasting workflows such as ARIMA-style baselines, rolling validation, and backtesting for structured temporal data.

## Included Here
- `forecasting_demo.py`: a runnable starter that builds a simple forecasting workflow and reports evaluation output
- `README.md`: the quick-start guide for this topic folder

## Run
`python forecasting_demo.py`

## Expand With
- ARIMA, SARIMA, and SARIMAX notebooks
- Prophet-style workflows and horizon-aware error analysis
