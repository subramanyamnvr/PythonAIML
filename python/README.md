# Python

## Overview
This folder contains Python fundamentals, essential libraries, and project examples to build a strong foundation in Python programming for AI/ML applications.

## Topics Covered
- Python basics (data types, control structures, functions)
- Essential libraries (NumPy, Pandas, Matplotlib)
- Project examples and best practices

## Prerequisites
- Python 3.9+

## Quick Start
```bash
pip install -r requirements.txt
```

## Subfolders
- **basics/** - Python fundamentals and syntax
- **libraries/** - Key Python libraries for data science
- **projects/** - Python project examples

## Learning Path
1. Start with basics/ for Python syntax
2. Learn libraries/ for data manipulation
3. Apply knowledge in projects/
