# Generative AI

## Overview
Explore generative AI technologies, including Large Language Models (LLMs), prompt engineering, and fine-tuning techniques.

## Topics Covered
- LLM fundamentals
- Prompt engineering
- Model fine-tuning
- Generative applications

## Prerequisites
- Deep learning basics
- Python programming
- Understanding of transformers

## Subfolders
- **llms/** - Large language models
- **prompt-engineering/** - Effective prompting techniques
- **fine-tuning/** - Model customization and training

## Learning Path
1. Understand llms/ architecture
2. Master prompt-engineering/ skills
3. Learn fine-tuning/ techniques
