# Computer Vision

## Overview
Computer vision techniques for image processing, object detection, and visual understanding using traditional and deep learning approaches.

## Topics Covered
- Image processing fundamentals
- Feature extraction
- Object detection and recognition
- Image segmentation

## Prerequisites
- Python programming
- Basic mathematics
- Understanding of ML concepts

## Subfolders
- **computer-vision/** - Core CV algorithms and techniques
- **image-processing/** - Image manipulation and preprocessing

## Learning Path
1. Learn image-processing/ basics
2. Apply computer-vision/ techniques
