GenAI Roadmap

Move from LLM basics into retrieval, tuning, orchestration, and platforms.

The GenAI folder now has a clean numbered first layer. Start with the lower numbers for foundations, then branch into retrieval, orchestration, PEFT, evals, multimodal work, or safety-oriented production topics depending on what you want to build next.

README Module map Start at 01 Jump to orchestration

Stage 1: Foundations

01 to 03

LLM basics, LangChain, OpenAI, and Ollama.

01-LLM-FoundationsHigh-level notes on LLM evolution and the transformer era. 02-LangChain-Basics-and-EcosystemCore LangChain workflow, ingestion, embeddings, and vector stores. 03-OpenAI-and-OllamaOpenAI and Ollama starter lessons plus chatbot examples.

Stage 2: Retrieval

04 to 07

RAG, conversation-aware retrieval, search, hybrid retrieval, and graph or SQL-backed knowledge work.

04-RAG-Document-QADocument-question answering and retrieval setup. 05-Conversational-RAGConversation-aware retrieval workflows. 06-Search-and-Hybrid-RetrievalSearch-engine style flows and hybrid retrieval experiments. 07-Structured-Data-and-Knowledge-GraphsChat with SQL and graph-backed reasoning paths.

Stage 3: Application Outputs

08

Summarization and targeted GenAI apps.

08-Summarization-and-Task-AppsText summarization and task-specific app examples such as MathsGPT.

Stage 4: Model Adaptation

09 to 10

Open-source models, quantization, and fine-tuning foundations.

09-Hugging-Face-and-Open-Source-ModelsHugging Face, CodeLlama, and compact-model notes. 10-Fine-TuningFine-tuning scripts and model adaptation notebooks.

Stage 5: Orchestration and Platforms

11 to 14

LCEL, orchestration patterns, cloud platform material, and extra framework references.

11-LCEL-and-LangChain-UpdatesLCEL, serve/client examples, and newer LangChain structures. 12-LangGraph-and-Orchestration-NotesWorkflow orchestration and LangGraph-oriented notebook examples. 13-Cloud-GenAI-PlatformsAWS Bedrock, Nvidia NIM, and cloud GenAI material. 14-Agentic-Framework-ExtrasExtra CrewAI and MCP-adjacent notes that were already in this area.

Stage 6: Advanced Production Topics

15 to 19

PEFT, evals, guardrails, multimodal work, and stateful agent design.

15-PEFT-LoRA-QLoRA-and-Inference-ServingEfficient fine-tuning and deployment-friendly serving patterns. 16-LLM-Evals-and-ObservabilityEval design, tracing, regressions, and runtime visibility. 17-Guardrails-and-AI-SecurityPolicy enforcement, validation, and threat-aware GenAI design. 18-Multimodal-GenAIImage, audio, and mixed-modality application patterns. 19-Agent-Memory-and-StateMemory stores, checkpoints, and state-aware agent workflows.

Stage 7: Legacy Background

20

Older neural-network notes kept for reference, not as the main GenAI path.

20-Legacy-Neural-Network-BackgroundANN, RNN, LSTM, and older sequence-model reference material grouped cleanly.

Keep in mind

Topic first

The root is now a readable topic list instead of a mix of loose PDFs, notebooks, and imported repos.

Imported repos

Platform and framework mini-projects stay intact inside the numbered folders when they carry their own app structure.

Fast route

If you already know LangChain basics, jump directly to the retrieval or fine-tuning stages and loop back later.

Legacy notes

The last folder is there for cleanup and preservation, not because it is the best next step for GenAI study.
