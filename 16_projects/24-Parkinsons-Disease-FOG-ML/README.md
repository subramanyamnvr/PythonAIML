# Parkinsons Disease FOG ML

## Overview
This project folder keeps the imported freezing-of-gait work visible from the main portfolio index while preserving the original source layout inside a nested directory.

## Included Here
- `Parkinson-s-Disease-FOG--main/lgbm-for-both-defog-tdcsfog.ipynb`: the main modeling notebook
- `Parkinson-s-Disease-FOG--main/README.md`: the original project notes from the imported source
- reference PDFs inside `Parkinson-s-Disease-FOG--main/`: background material for the approach
- `requirements.txt`: a conservative dependency list for the notebook workflow

## Run
1. Install dependencies with `pip install -r requirements.txt`.
2. Open `Parkinson-s-Disease-FOG--main/lgbm-for-both-defog-tdcsfog.ipynb`.
3. Keep the nested folder structure intact so notebook-relative paths continue to work.

## Notes
- The root folder is intentionally thin and acts as a clean wrapper around the imported project.
- If you later flatten this project, update this README first so the portfolio index stays accurate.
