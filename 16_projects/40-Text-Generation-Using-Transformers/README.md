# Text Generation Using Transformers

## Overview
This folder contains a notebook-based text-generation exercise built around the `daily_dialog` dataset and a transformer fine-tuning workflow.

## Included Here
- `Text_Generation.ipynb`: the main notebook for preprocessing, fine-tuning, and evaluation
- `input.txt`: an exported sample text file used during experimentation
- `requirements.txt`: a conservative dependency list for rerunning the notebook locally

## Run
1. Install the dependencies with `pip install -r requirements.txt`.
2. Open `Text_Generation.ipynb` in Jupyter.
3. Follow the notebook cells in order to preprocess data, fine-tune, and evaluate perplexity.
