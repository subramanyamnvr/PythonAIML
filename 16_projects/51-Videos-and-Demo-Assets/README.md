# Videos And Demo Assets

## Overview
This folder stores demo videos that support computer-vision and safety-monitoring projects in the portfolio.

## Included Here
- `bikes.mp4`, `cars.mp4`, `motorbikes.mp4`, `people.mp4`: motion samples for detection/counting demos
- `ppe-1.mp4`, `ppe-2.mp4`, `ppe-3.mp4`: PPE-related demo clips
- `requirements.txt`: intentionally empty because this folder is asset-only

## Notes
- There is no direct Python entrypoint in this folder.
- Keep this folder as a media library and reference the files from project-level scripts when needed.
