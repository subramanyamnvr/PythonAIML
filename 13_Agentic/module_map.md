# Agentic Module Map

This map shows the cleaned first-layer sequence for the agentic folder.

## Main Course Path

- `01-Foundations`  
  Core foundations labs and baseline agentic patterns.

- `02-OpenAI-Agents-SDK`  
  OpenAI Agents SDK material and related week-two exercises.

- `03-CrewAI`  
  CrewAI projects, workflows, and community extensions.

- `04-LangGraph`  
  LangGraph workflows, graph-based orchestration, and related builds.

- `05-AutoGen`  
  AutoGen agent patterns and community variations.

- `06-MCP`  
  Model Context Protocol labs, demos, server patterns, and community projects.

## Support Areas

- `assets`  
  Shared images used by notebooks and the top-level README.

- `guides`  
  Supporting guides for Python, APIs, Git, notebooks, async work, and project setup.

- `setup`  
  Platform-specific environment setup instructions and troubleshooting notebooks.

## Layout Notes

- The first layer is still flat, but the week folders now use consistent two-digit numbering.
- Support folders were left in place because the labs and README already link to them directly.
- Internal references were updated so the renamed week folders continue to work with the existing course material.
