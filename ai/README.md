# Artificial Intelligence

## Overview
Explore the foundations of Artificial Intelligence, including core concepts, algorithms, and problem-solving techniques.

## Topics Covered
- AI fundamentals and history
- Search algorithms
- Knowledge representation
- Machine learning basics
- Ethics in AI

## Prerequisites
- Basic programming knowledge
- Mathematics (logic, probability)

## Subfolders
- **fundamentals/** - Core AI concepts and theory
- **algorithms/** - AI algorithms and implementations

## Learning Path
1. Understand fundamentals/ concepts
2. Implement algorithms/ in code
