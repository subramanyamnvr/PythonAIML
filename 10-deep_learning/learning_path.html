Deep Learning Roadmap

Start at the top and move stage by stage.

The deep learning folder is flat at the first level now. Open the lowest-numbered folder in the stage you want, then run the notebooks inside it in project order. The cleaned notebooks use a portable setup cell so they work more reliably from the folder itself or from the repo root.

Keep in mind

Notebook order

Inside a folder, follow the notebooks in `Project_01`, `Project_02`, `Project_03` order whenever that sequence exists.

Dataset folders

`Circle_Cross`, `CatDog`, `WESAD_data`, and similar trees stay nested on purpose because the notebooks read them as structured datasets.

Portable setup

The notebooks that previously jumped to a hardcoded Colab drive path now start with a folder-aware setup cell.

Fast path

If you already know the basics, jump straight from Stage 2 into Stage 3 or Stage 4, then return for the missing fundamentals later.
